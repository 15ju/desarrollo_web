Modelo cliente-servidor

Cuando en el navegador ingresamos una URL, nos lleva a donde esta guardado el codigo fuente del sitio al que estoy ingresando

En la consola de navegador de google chrome esta la solapa network que nos muestra todas las solicitudes que se hicieron cuando le pedi que ingrese a una pagina web

Para que un sitio quede publicado tengo que subirlo a algun servidor, porque quiero que quede accesible a todo el mundo, si lo tengo en mi pc editado nadie mas puede entrar

Direcciones IP

Se esta trabajando con direcciones ipv4, pero se esta trabajando ya en ipv6. Cada servidor tiene una direccion ip diferente

Lo ideal es que se termine el proyecto final, que es subirlo a Github, que la url quede disponible para compartirla y que cualquiera pueda ingresar a verlo.

Dominio

En argentina, los dominios que son .com.ar se compran en NIC, si no tiene que ser .com.ar se pueden comprar en otros lugares(Ej. GoDaddy). Hay que registrar el dominio(comprarlo) para despues tener una web. El servidor almacena lo que se ve en ese dominio.
