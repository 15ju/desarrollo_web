Uso interno

Hay distintas formas de trabajar css, una es dentro del mismo documento HTML, en este caso va dentro del head, con etiquetas STYLE y dentro de eso todo lo que sea correspondiente a CSS

A este documento le puse el elemento p con color rojo y con el ancho definido en 300px

Esta manera de usar CSS dentro del documento HTML se denomina inline styles. Lo ideal seria que esta hoja de estilos se haga aparte para poder reutilizarlas en vez de hacer siempre una nueva.

Desarrollar CSS aparte

Corresponde desarrollar el css en una hoja externa para luego poder usarlo en varias paginas diferentes, por ejemplo, en una pagina donde se compra, si lleva a un carrito o algo asi, no va a haber que escribir pagina por pagina el CSS

Se escribe el CSS en una hoja aparte, algo.css, y se define ahi todo el estilo que se le va a dar, luego se va a referenciar al mismo en los distintos documentos HTML
