Mas de GIT

La zona de index es una zona virtual que maneja GIT para ir guardando los cambios

Mas comandos git:

  • git blame (nombre del archivo) me va a mostrar linea por linea como se modifico y quien lo hizo, en que momento tambien
  • Para subir un repositorio a github (lo busca en google) hay que:
    • se registra
    • git remote add origin URL utilizando la URL del proyecto al que este subiendo el contenido
    • git push -u origin master lo va a subir directamente

GIT no te deja hacer ningun commit si no te identificas con el usuario y mail

Trabajando con el repositorio local me puede pasar que se borre la carpeta local, lo que haria que el repositorio local desaparezca y se pierda todo lo que hice hasta ese momento

Para no perder los datos esta el repositorio remoto, como github, bitbucket u otro. Puedo perder contenido? si, si hay alguna modificacion que todavia no subi al repositorio remoto y, por ejemplo, se me rompe el ssd, perderia contenido pero no todo ya que solo se dejaria de tener el que no se haya subido al repositorio remoto
