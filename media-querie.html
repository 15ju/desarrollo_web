Media queries

Se utilizan para establecer parametros que digan cuando debera cambiar el formato de la pantalla que se diseñó. Ejemplo: se pone @media y se establece que consulte (querie) la dimension de la pantalla para saber si es mayor que una cierta cantidad de pixeles, en caso contrario, va a cambiar algo

Es importante saber que es viewport, media querie y saber que existen librerias para poder tener los conocimientos teoricos que son requeridos para realizar una pagina full responsive

Librerias full responsive

Se busca en bootstrap o alguna otra libreria para usar el diseño full responsive que ya este hecho
